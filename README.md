## üìä How to Use

Now that you have the application, here are some commands you can use to analyze stocks.

### 1. Full Analysis of a Single Stock
Use this command for in-depth research on a specific stock:
```
/full-analysis [TICKER]
```
**Example:** `/full-analysis RCAT`
- Gathers real-time quotes, financial reports, and institutional holdings.
- Checks for catalysts (PDUFA dates, earnings reports).
- Computes a risk score.
- Provides entry and exit suggestions.
- Automatically generates a detailed report.

### 2. Short-term Scan
When looking for promising trading opportunities, use:
```
/stock-scan
```
- Scans day's top-performing stocks and price changes.
- Filters out low-quality stocks.
- Produces a "short-term scan" report.

### 3. Generate Summary Report
After analyzing multiple stocks, review your results:
```
/summary
```
- Reads all session data.
- Compares stocks based on priority and risk-reward ratio.
- Creates a timeline of catalysts.
- Outputs final trading decision recommendations.
